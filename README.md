###  Sobre mim

Tenho experiência em Desenvolvimento Full Stack e Administração de Sistemas. Trabalho com todo o ciclo de criação de aplicações, usando React e JavaScript no front-end e Node.js com APIs REST no back-end. Sou uma pessoa prática, responsável e com boa comunicação, sempre buscando entregar soluções claras e funcionais.
